# Data Directory

This directory contains raw and processed data files for the AquaSense project.

## Raw Data (data/raw/)

- **TN.csv**: Total Nitrogen measurements
- **TP.csv**: Total Phosphorus measurements
- **TN_STRF.csv**: Total Nitrogen stratified data
- **OP_STRF.csv**: Orthophosphate stratified data
- **NH3_STRF.csv**: Ammonia stratified data
- **SSC_STRF.csv**: Suspended Sediment Concentration stratified data

## Processed Data (data/processed/)

- **merged_dataset.csv**: Cleaned and merged dataset ready for model training
